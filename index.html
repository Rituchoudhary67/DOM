  1. getElementByTagName()
            
      
                
    • The getElementsByTagName() method returns a collection of all elements with a specified tag name.
      • 
                
    •  The getElementsByTagName() method returns an HTMLCollection.
      • 
                
    •  The getElementsByTagName() property is read-only.
      •  
                
    
            
    
               
                    Example:-
                Get all elements with the tag name "li":
                const collection = document.getElementsByTagName("li");

                getElementByClassName()
            
                    The getElementsByClassName method of Document interface returns an array-like
                object of all child elements which have all of the given class name(s).
                
    Syntax
     getElementsByClassName(names)
                
    NOTE
     A string representing the class name(s) to match; multiple class names 
                are separated by whitespace.
                Example:-document.getElementsByClassName('red test')
            
    
            
    
                Example:- 
                
    
                    LEARNING DOM SELECTOR
                
    
                const color = document.getElementsByClassName("red");
                console.log(color)

  3. getElementById()
                
                        1. The getElementById() method returns an element with a specified value.
                    2. The getElementById() method returns null if the element does not exist.
                    3. The getElementById() method is one of the most common methods in the HTML DOM. 
                       It is used almost every time you want to read or edit an HTML element.

                    Note:- Any id should be unique, but: 
                           If two or more elements with the same id exist, getElementById() returns the first. 
                             
                    EXAMPLE:- 
                    1. const element = document.getElementById("eleid")
                       console.log(element)
                    2. if i want to change the color by id name.
                       const myElement = document.getElementById("demo");
                       myElement.style.color = "red";

  4. querySelector()
            
                    The querySelector() method returns the first element that matches a CSS selector.
                To return all matches (not only the first), use the querySelectorAll() instead.
                Both querySelector() and querySelectorAll() throw a SYNTAX_ERR 
                exception if the selector(s) is invalid.

                Example:- 1.Get the first p element:
                            document.querySelector("p");
                          2.Get the first element with class="example":
                            document.querySelector(".example");  
            
    
            
    5. 
            
  5. querySelectorAll()
                
                        The Document method querySelectorAll() returns a static (not live) NodeList representing a list
                    of the document's elements that match the specified group of selectors.

                    Syntax:- querySelectorAll(selectors)

                    Example:- 1. Get the all p element: 
                                 const allelement = document.querySelectorAll("p");

                The Difference Between an HTMLCollection and a NodeList
                     

                        Node list
                        HTML collection
                     

                     

                        A NodeList is a collection of document nodes(element nodes, attribute nodes, and text nodes)
                        An HTMLCollection is a collection of document elements.
                     

                     

                        NodeList items can only be accessed by their index number.
                        HTMLCollection items can be accessed by their name, id, or index number.
                     

                     

                        A NodeList is most often a static collection. 
                        An HTMLCollection is always a live collection.
